Rock paper scissorsmore_vert
Rock paper scissorsclose

Rock paper scissors is a tutorial project. This project is using JS math() methods, math floor and random, to generate random computer choice. Switch loop is comparing between user and computer choice, and depending of the outcome, result component is refreshed, and classlist.add and classlist.remove is used to make different color glow on buttons.

Memory Card Gamemore_vert
Memory Card Gameclose

Memory Card Game - is a JavaScript Tutorial. This project is selecting elements in the DOM with querySelector, using forEach() method to iterate through list, adding and removing an element classes and event listeners, using css grid to position an items, and using perspective and backface transition to simulate card flip. To shuffle cards for every new game, math floor and math random was used.

Markdown previewclose

As a part of Free Code Camp front end libraries projects was to build a drum machine. This project is using keydown event listener, a JS higher order function map() to loop through object containing keydown, keyCodes and links to audio source for html audio tag to reproduce a specific sound. To prevent all kinds of bugs, add and remove event listener was placed on life cycles hooks DidMount and WillUnmount.
